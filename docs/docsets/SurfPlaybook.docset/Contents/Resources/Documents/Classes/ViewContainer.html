ViewContainer

public class ViewContainer<View> : UIView where View : UIView

Вспомогательный контейнер-view, позвооляет обернуть UI-компонент типа UIView в таблицу, чтобы показать его в рамках playbook-а.

  • Reference: Причина возникновения и проблема, которую решает контейнер, а также решение описаны в источнике

    height
