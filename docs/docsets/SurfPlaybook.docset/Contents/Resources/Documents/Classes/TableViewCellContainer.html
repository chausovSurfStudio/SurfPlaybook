TableViewCellContainer

public class TableViewCellContainer<Cell> : UIView, UITableViewDataSource, UITableViewDelegate where Cell : UITableViewCell

Вспомогательный контейнер-таблица, позвооляет обернуть UI-компонент типа UITableViewCell в таблицу, чтобы показать его в рамках playbook-а.

  • Reference: Причина возникновения и проблема, которую решает контейнер, а также решение описаны в источнике

    heightResolver

    Замыкание, которое вызывается в методе tableView:heightForRowAt:, позволяет расчитать и вернуть высоту ячейки. В случае передачи nil используется UITableView.automaticDimension. По-умолчанию равно nil.
