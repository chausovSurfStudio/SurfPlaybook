PlaybookChapter

public final class PlaybookChapter

Отдельная глава Playbook-а.

Каждой главе соответствует своя карусель на главной странице. Представляет собой набор страниц, PlaybookPage, где каждой странице соответствует свой reuse-компонент, который необходимо отобразить в рамках playbook-а.
